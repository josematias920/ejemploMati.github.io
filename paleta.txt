<!-- Formularios-->
<form>    contiene todos los elementos htm de un Formulario
action    indica a que url se enviara la informacion
metod     indica si la peticion sera GET o POS
<input>   permite introducir texto
type      es un elemento obligatorio en el que se señala el tipo de texto que se espera obtener en input
id        es obligatorio si en label hay un atributo for, este ocupa un id de la pagina
name      opcional, nombre asignado a la peticion HTTP
value     opcional, sera el valor de la peticion HTTP cuando sea enviada
required  hace que el campo siempre este lleno
readonly  solo lectura
pleaceholder  texto de ayuda
value     valor por defecto en el campo
<select>  genera una lista desplegable
<option>  opciones que estaran en select
<textarea>  genera el texto para descripciones
<label>   etiquetas de texto para formulario
for       tiene la misma etiqueta del id de input, select o textarea
<button>    genera un boton
type      comportamiento del boton, este puede ser submit, reset, button
